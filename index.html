Function syntax in Javascript

                    function foo(bar, baz) {
                        var rez = bar + baz;
                        return rez;
                    }

Same thing in Scala...

                    def foo(bar: Int, baz: Int) {
                        val rez = bar + baz;
                        rez;
                    }
